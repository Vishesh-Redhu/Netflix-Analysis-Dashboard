An enterprise-grade dashboard designed to analyze 8,800+ Netflix records and provide AI-powered content recommendations.

---

## 🚀 Features
* **🤖 Smart Recommendations:** Content-based filtering using **TF-IDF Vectorization** and **Cosine Similarity**.
* **💡 AI Insights:** Automated data storytelling that updates based on your active filters.
* **🔬 Deep Dive Analytics:** * Interactive Sunburst charts for Genre/Rating hierarchy.
    * Thematic Word Clouds based on content descriptions.
    * Monthly release velocity heatmaps.
* **✨ Premium UI:** Modern **Glassmorphism** design with a cinematic dark theme.
